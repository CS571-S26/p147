# StudySpotter — UW Study Space Finder

A React web app that helps UW–Madison students quickly find a good place to study based on what they need right now.

**Live site:** https://cs571-s26.github.io/p147/

## Features

- Search by name, type, or amenity
- Filter by noise level and amenities
- Sort by name or occupancy (least/most busy)
- Live occupancy meters with check-in/check-out
- Save favorites (persisted in localStorage)
- Comments on each location (persisted in localStorage)
- Export/import favorites as JSON

## Running Locally

```bash
npm install
npm run dev
```

## Deploying

```bash
npm run deploy
```
